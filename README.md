# MyWallet - CATAN.IA AI Student Challenge 2026

Progetto sviluppato per la **Traccia 1**.

## 📋 Descrizione
**MyWallet** è una Web App di finanza personale pensata per aiutare i giovani a gestire il proprio budget monitorando entrate e uscite in modo semplice e intuitivo.


## 🛠️ Stack Tecnologico
Per la realizzazione di questo progetto sono state adoperate le seguenti tecnologie:
* **Backend**: Python.
* **Frontend**: JavaScript (Localhost:3000), HTML e CSS.
* **DB**: Supabase.


## 🚀 Come avviare l'app
Per avviare l'app in locale è necessario eseguire i seguenti step:

### 1. Clonazione:
   
   È necessario scaricare i file sul proprio dispositivo clonando la repository tramite terminale:
   ```bash
   git clone https://github.com/ENOMIS007/MyWallet.git
   ```
   Successivamente spostarsi dentro la cartella appena clonata:
   ```bash
   cd MyWallet
   ```


### 2. Installazione dipendenze:
   
   Installare tutte le dipendenze tramite i comandi:
   ```bash
   pip install -r requirements.txt
   npm install
   ```

   
### 3. Configurazione:
   
   Rinominare manualmente `.env.example` in `.env` e inserire le chiavi Supabase (URL e Anon Key).

   
### 4. Avvio Server:
   
   Dal terminale, spostarsi dentro la cartella MyWallet\backend:
   ```bash
   cd backend
   ```
   Avviare il server tramite il comando:
   ```bash
   python main.py
   ```

   
### 5. Esecuzione:
   
   Una volta avviato il backend, aprire il browser e digitare:
   ```bash
   localhost:3000
   ```
